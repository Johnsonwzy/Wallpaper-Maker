# Changelog

本项目变更记录采用"已新增 / 已优化 / 已修复"分组，按时间倒序维护。

## [2026-04-03] P0 建议落地：测试扩展 + 性能预警

### 已新增

- pytest 扩展至 70 用例（新增 `tests/test_layouts_export_filters.py`）：覆盖 14 种布局组合、5 种导出格式边界、12 滤镜极端强度、7 叠层极端强度、布局辅助函数、公开 API 入口。
- UI 性能预警标签：当分辨率/张数/批次组合可能导致高耗时或高内存时，在数值面板下方实时显示提示（含预估耗时和内存峰值）。
- 性能预警函数 `estimate_perf_hint()` 加入 `gui_utils.py`，含 3 项单元测试。

## [2026-04-03] 深度模块化重构

### 已新增

- 完成 10 轮功能域切片，从 2 个大文件扩展为 25 个职责清晰的模块。
- 新增 GUI 子模块：`app_build_ui`（UI 构建）、`app_preview_pipeline`（预览刷新）、`app_export_jobs`（导出编排）、`app_presets_io`（预设 I/O）、`app_windowing`（窗口几何）。
- 新增核心子模块：`core_layouts`（30+ 布局渲染器）、`core_filters`（12 类滤镜）、`core_overlays`（背景叠层）、`core_export`（导出编码）。
- 新增工具模块：`style_intensity`（强度映射）、`skip_stats`（跳过统计）。

### 已优化

- `app.py` 从 ~3 100 行瘦身至 1 068 行，保留薄委托层角色。
- `core.py` 从 ~2 500 行瘦身至 863 行，保留渲染编排入口角色。
- 模块间无循环依赖，GUI 子模块通过 `app` 实例参数松耦合。
- 模块平均行数降至 334 行，可维护性评分从 7.2 提升至 8.6。

## [2026-04-03] 预览与稳定性增强版本

### 已新增

- 新增预览"对比增强模式"（仅预览生效，不影响正式导出）。
- 新增 GUI 自检入口：`python3 -m wallpaper_maker.main --self-test`。
- 新增 GUI 自检扩展参数：`--self-test-verbose`、`--self-test-no-preview`。
- 新增文档：`README.md`、`项目评估报告.md`、`CHANGELOG.md`。

### 已优化

- 滤镜与叠层参数重检并增强，中档强度下差异可见性提升。
- 叠层/滤镜推荐强度整体上调，减少来回试错。
- 主窗口与预览画廊默认几何参数按 Studio Display 场景优化。
- 预览画廊与主窗口尺寸同步策略加强（同尺寸、对称、贴边）。
- "跟随系统"主题模式支持自动轮询刷新，无需手动重复选择。

### 已修复

- 修复预览窗初始化异常：错误调用 `winfo_minsize` 导致 Tk 回调崩溃。
- 修复退出期潜在回调问题：主窗口关闭时主动取消定时任务并销毁预览窗。
- 修复叠层切换后预览不实时更新问题。
- 修复部分导入/符号遗漏引发的运行时错误（如默认布局常量、重复函数干扰等）。

## [2026-03] 模块化与能力扩展阶段

### 已新增

- 由单文件脚本升级为包结构：`wallpaper_maker/`。
- 新增标准入口：`python3 -m wallpaper_maker`（`__main__.py`）。
- 新增背景混合架构：`bg_base_style` + `bg_overlay_style` + 强度控制。
- 新增多种高级叠层：磨砂、暗角、几何纹理、纸张纹理、柔光雾化、电影分级、中心聚光等。
- 新增 12 类滤镜（含 Lomo 系列）及强度控制。
- 新增单图预览画廊（缩放、前后/首末导航、1:1、空格快比）。
- 新增"将当前滤镜批量应用到现有预览并导出"能力。

### 已优化

- 支持预览缓存与快速重渲染：切滤镜/叠层/底色不必全量重排版。
- 默认图源与配置常量集中管理（`config.py`）。
- 预设结构升级并兼容旧字段（含 `bg_style` 到新架构映射）。

### 已修复

- 解决 `python -m wallpaper_maker.main` 的 runpy 预导入告警问题（懒加载导出 + 标准入口）。
- 解决样式强度栈定义缺失导致的运行时报错。
- 清理重复实现与若干名称解析错误，提升运行稳定性。
